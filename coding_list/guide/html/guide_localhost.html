Localhost 서버 구동 기초가이드

1. 명령프롬프트 실행    // WIN+R 누른 후 cmd 입력 후 엔터

2. 프로젝트 루트 경로로 이동    // 예 cd C:\work\front

3. npm run start 입력

4. Server 스타트 메시지 확인 후 http://localhost/ 접속

5. Server 종료 Ctrl + C 후 메시지 나오면 Y 입력 후 엔터

6. scss 및 js 전체 빌드 명령어 : npn run build    // 서버구동중에는 자동 빌드 되나 이와 별도로 따로 빌드가 필요할 경우 사용.
