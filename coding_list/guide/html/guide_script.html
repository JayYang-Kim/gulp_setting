스크립트 공통 가이드

기본사항
  1. 기본 구조는 부트스트랩 JS 구조를 따랏으며 jquery, ES6 문법이 섞여 있습니다.

폴더구조

  • /etc.clientlibs/yonsei/clientlibs/common/clientlib-site/assets/js/ -- 1. 기본 경로
  • /etc.clientlibs/yonsei/clientlibs/common/clientlib-site/assets/js/src/ -- 2. 모듈 리스트
  • /etc.clientlibs/yonsei/clientlibs/common/vendor/ -- 3. 외부 플러그인 리스트
  • /etc.clientlibs/yonsei/clientlibs/common/clientlib-site/site/ -- 4. babel로 빌드된 js파일 경로

1. 기본 경로

common

2. 모듈 리스트

cardUi, customAlert, draggble, event, form, jumpTo, layout, scrollTab, treeMenu, util

3. 외부 플러그인 리스트

bootstrap, bootstrap-datetimepicker, chart, ckeditor, jquery, masonry, mCustomScrollbar, slick, transit

* 플러그인 호출 방법 예) cardUi

		1. data 속성을 통한 호출, 옵션 필요 시 data-options 속성을 이용하여 JSON 데이터 형식으로 추가
		

	
		



		2. 직접 호출
		$('selector').cardUi({
			options
		});

위 방식을 이용하여 호출 할 수 있는 플러그인 리스트

mCustomScrollbar, slick, bootstrap-datetimepicker, cardUi, scrollTab, treeMenu

비동기로 추가된 DOM에 대한 플러그인 재 호출 메서드

			$('body').LayoutAction('initLibray');

특정 해상도에서만 플러그인 호출이 필요할 경우 enableMode 옵션 추가
