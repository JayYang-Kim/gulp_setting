  • 중복되는 파일명에 대해서는 SITE 정의서 내 이전 depth의 파일명 추가 기입
  • 프로세스가 있거나, 한 화면에 여러가지 케이스가 있는 페이지는 뒤에 번호 추가 기입( 예 : /etc.clientlibs/sev/department/department_01.html, /etc.clientlibs/sev/department/department_02.html)
