Custom Alert

		
		YUHS.customAlert()

Options

  • type : alert(default), confirm //메세지 박스 타입 설정
  • message : ""(default) //노출 될 메세지 문자열
  • defaultBtn : '<button type="button" id="customMsgSubmit" class="btn btn-primary" data-dismiss="modal">확인<button>'(default) // 기본 버튼 명
  • cancelBtn : '<button type="button" id="customMsgCancel" class="btn btn-success" data-dismiss="modal">취소</button>'(default) // 취소 버튼

Callback

  • defaultCallback : 확인 버튼 클릭 시
  • cancelCallback : 취소 버튼 클릭 시

Example

  • 퍼블리싱 가이드 > 커스텀 얼럿, 컴폼
