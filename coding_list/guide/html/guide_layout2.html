layout

공통 부분 전반적인 기능 모음(Header, Footer 등 공통 UI) -- 화면 로딩 후 자동 실행

options

없음

Method

  • $('body').LayoutAction('initLibray'); -- 비동기로 추가된 DOM에 대한 플러그인 재 호출
  • $('body').LayoutAction('footerFixed'); -- Footer Fixed Check
  • $('body').LayoutAction('commonBanner'); -- PC, Tablet 상단 띠베너 컨트롤 메서드(띠베너 관련태그 삽입 후 호출)
  • $('body').LayoutAction('setFooterPadding'); -- 모바일 화면 하단에 스티킹되는 컨텐츠 삽입 후 여백 조정 메서드(푸터 부분에 스티킹 되는 컨텐츠 삽입 후 호출)
