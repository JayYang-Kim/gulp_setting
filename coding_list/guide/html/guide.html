대표적인 gulp.js 사용 모듈

gulp-babel

로컬 라이브서버
gulp-connect-ssi      
connect 라이브서버에서 SSI를 사용 가능하게 해주는 플러그인
include 기본 세팅은 아파치 SSI 환경으로 설정되어 있습니다. 관련 링크

기준 해상도

PC         : 최소 1024px 이상

Tablet   : 768px ~ 1024px

Mobile  : 360px ~ 767px

PC

MAC : Safari 최신버전 ※ 애플社에서 2012년 10월이후 윈도우 Safari 지원안함
Tablet / Mobile

프로젝트 경로(Front)


html      
/Root/html/

html include      * html 내에 공통 요소들 예) top, header, footer 등
/Root/html/includes/

scss 공통 요소      * 프론트단 모든 scss파일에서 공유하는 변수, 믹스인, 함수 등 모음, 외부 라이브러리 css
/Root/static/asset/common_scss/

scss      
/Root/static/asset/scss/

css      * scss 생성 및 수정 작업 후 저장 시 자동으로 컴파일되어 같은 이름의 css파일(min)로 생성됩니다. 직접수정 금지(수정은 scss파일을 통해서만 진행 해주세요.)
/Root/static/di/css/

js 작성용      * 수정은 이쪽 파일에서 진행 해주세요.(ES6문법 사용 가능)
/Root/static/asset/js/

js 배포용      * 작업 후 저장 시 자동으로 컴파일되어 같은 이름의 js파일(min)로 생성됩니다.
/Root/static/di/js/

images      
/Root/static/di/images/

font      
/Root/static/di/fonts/
